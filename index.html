🧠 Pourquoi « Codex Mental » ?

Codex Mental est un espace personnel, un laboratoire numérique où pensées, intuitions, observations et systèmes se rassemblent.

Le mot Codex évoque un manuscrit ancien, soigneusement organisé, parfois ésotérique, parfois technique. Il suggère un savoir structuré, personnel, transmissible, mais aussi mystérieux ou sacré.

Mental, c’est l’univers intérieur : la mémoire, la pensée arborescente, les obsessions créatives, les constellations de sens qu’on ne peut pas toujours exprimer, mais qu’on peut cartographier.

Codex Mental est donc un journal vivant, à mi-chemin entre carnet d’observation, bibliothèque mentale et grimoire numérique. C’est là que se déposent les synchronicités, les fragments d’idées, les outils, les expériences – non pas pour les figer, mais pour leur donner une forme évolutive.
